List past conversation sessions for the current project with metadata.

## Usage notes

- Returns session ID, title, message count, and last update time
- Sessions are sorted by most recently updated first
- Use the session ID with get_session_history to read a past conversation
- Useful for finding previous solutions, recalling context, or reviewing past work
